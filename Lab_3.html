Preguntas realizadas:

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?
Evitar usar !important en CSS, ya que puede complicar el mantenimiento y depuración del código
Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?
Elige las imágenes de fondo con tamaños de archivo pequeños para asegurar tiempos de carga
Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?
Usar "%" para diseños responsivos, "px" para control exacto del tamaño y "pt" para medidas de texto, especialmente en impresión
¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?
La versión minimizada del css reduce el tamaño del archivo, eliminando espacios y comentarios, lo que acelera el tiempo de carga de la pagina.
